# FinEdge Engine Group 10

FinEdge personal finance tracker using Node.js and Express.

## Description

This project implements a financial engine with Express.js, providing API endpoints for health checks and other financial operations.

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies

## Usage

- Start the development server: `npm run dev`
- Start the production server: `npm start`

## API Endpoints

- `GET /health` - Health check endpoint

## Dependencies

- Express.js - Web framework
- Nodemon - Development tool for auto-restarting
